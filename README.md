## 이 책에 대하여
이 책은 머신러닝(ML) 팀이 효과적으로 제품을 만들고 운영하기 위한 실천 전략을 다룹니다. 린 기반의 ML 개발 방식부터 MLOps, CI/CD, 자동 테스트, 컨테이너 환경 구성, 리팩터링 기법까지 폭넓게 설명하며, 협업 구조와 팀 운영 방식, 조직 내 확장 전략까지 함께 제시합니다.
